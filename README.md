# DeskTimer

A simple Windows desktop timer designed to help you track your working hours and know when to take a break.

## Features
- Set a timer by hours and minutes
- Pause, resume, or stop the timer at any time
- Timer stays on top of all other windows so you can always see it

## Requirements
- Windows OS

## How to Use
1. Enter your desired hours and minutes
2. Click **Start** to begin the timer
3. Use **Pause** and **Resume** to control the timer
4. Click **Stop** to cancel the timer at any time
5. The timer will notify you when time is up
